# ChatGPT 결과 — B2B 제조기업 홈페이지 리뉴얼 프로젝트 킥오프 회의록

## 1. 회의 목적

기존 홈페이지의 모바일 가독성 저하와 제품 문의 전환율 저하 문제를 개선하기 위해 홈페이지 리뉴얼 방향, 1차 구축 범위, 주요 기능, 자료 전달 일정, 후속 작업 일정을 논의했습니다.

---

## 2. 결정사항

* 리뉴얼 1차 목표는 다음과 같이 정리했습니다.

  * 회사 신뢰도 개선
  * 제품 정보 탐색 편의성 향상
  * 문의 접수 증가
* 주요 메뉴는 다음으로 구성하기로 했습니다.

  * 회사소개
  * 제품소개
  * 기술자료
  * 납품사례
  * 고객문의
* 제품소개 페이지에는 제품군별 목록, 상세페이지, 인증서 다운로드 영역을 포함하기로 했습니다.
* 고객문의 페이지에는 이름, 연락처, 이메일, 문의 제품군, 문의 내용을 입력받는 폼을 구성하기로 했습니다.
* 문의 폼 접수 내역을 이메일로 발송하는 기능은 기본 범위에 포함 가능합니다.
* 1차 범위에는 관리자 페이지 내 제품 정보 수정 기능만 포함하는 방향으로 제안되었습니다.
* 인증서 파일 업로드 관리 기능은 추가 견적 대상으로 분리하는 방향으로 제안되었습니다.
* 다음 회의는 사이트맵 초안 작성 후 진행하기로 했습니다.

## 4. 확인된 사실

* 기존 홈페이지는 오래되어 모바일 화면에서 가독성이 떨어집니다.
* 기존 홈페이지의 제품 문의 전환율이 낮다는 문제가 제기되었습니다.
* 클라이언트는 관리자 페이지에서 제품 정보를 직접 수정하기를 원합니다.
* 제품 정보 관리 기능은 구현 가능합니다.
* 인증서 파일 업로드 관리까지 포함할 경우 개발 범위와 일정이 늘어날 수 있습니다.
* 기존 홈페이지 게시판 데이터 이전 여부는 아직 확정되지 않았습니다.
* 게시판 데이터 양과 구조는 현재 확인되지 않았습니다.
* 전체 오픈 목표일은 다음 달 말로 논의되었습니다.
* 최종 일정은 자료 전달일과 추가 기능 범위 확정 후 다시 산정하기로 했습니다.
* 다국어 페이지 필요 가능성이 언급되었으나, 언어 범위와 번역 방식은 정해지지 않았습니다.
