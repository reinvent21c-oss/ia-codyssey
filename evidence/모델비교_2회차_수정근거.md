# 모델 비교 2회차 수정 근거

## 1. 수정 배경

1회차에서는 ChatGPT 5.5, Claude Opus 4.8, Gemini 3.5 Flash를 비교했다.

하지만 Gemini 3.5 Flash는 빠른 응답과 경량 작업에 강점이 있는 모델로, ChatGPT 5.5 및 Claude Opus 4.8과 동일한 등급의 상위 문서화/추론 모델로 보기에는 한계가 있었다.

따라서 2회차에서는 Gemini 3.1 Pro를 추가 테스트하여 모델 비교의 공정성을 보완했다.

## 2. 테스트 방식

Gemini 3.1 Pro 테스트는 기존 Gemini 3.5 Flash 대화의 영향을 줄이기 위해 새 대화창에서 진행했다.

테스트 조건은 다음과 같다.

- 동일한 회의 메모 사용
- 동일한 출력 항목 사용
- 동일한 작성 규칙 사용
- 이전 Gemini 3.5 Flash 결과 미참고
- 다른 모델의 답변 결과 미참고
- 평가 기준은 정확성, 결정사항/미확정 사항 분리, 후속 실행 항목 정리, 형식 준수, 실무 활용 가능성으로 설정

## 3. 평가 기준

| 평가 항목 | 확인 기준 |
|---|---|
| 정확성 | 회의 메모에 없는 내용을 확정적으로 작성하지 않았는가 |
| 결정사항/미확정 사항 분리 | 확정된 내용과 검토 중인 내용을 구분했는가 |
| 후속 실행 항목 정리 | 담당 역할, 해야 할 일, 기한/상태가 구분되었는가 |
| 형식 준수 | 요청한 7개 출력 항목을 지켰는가 |
| 실무 활용 가능성 | 웹에이전시 PM이 회의록 초안으로 바로 활용할 수 있는가 |

## 4. Gemini 3.1 Pro 테스트 결과 요약

Gemini 3.1 Pro는 기존 Gemini 3.5 Flash보다 문서 구조화와 맥락 이해 측면에서 개선된 결과를 보였다.

특히 출력 항목을 비교적 안정적으로 구분했고, 사내 공유용 회의록 형식도 유지했다.

다만 이번 과업의 핵심은 단순 요약이 아니라 결정사항, 검토사항, 확인 필요 사항을 안정적으로 분리하는 것이었다. 이 기준에서는 Claude Opus 4.8이 가장 일관된 결과를 보였다.

## 5. 2회차 반영 내용

2회차에서는 다음 사항을 보완했다.

| 구분 | 1회차 | 2회차 보완 |
|---|---|---|
| Gemini 비교 모델 | Gemini 3.5 Flash | Gemini 3.1 Pro |
| 비교 한계 | Flash 계열 모델이라 상위 모델 간 비교로 보기 어려움 | Pro 계열 모델을 추가 테스트하여 비교 공정성 보완 |
| 모델 비교 방식 | 결과 중심 비교 | 모델 등급의 적절성까지 함께 검토 |
| 최종 모델 | Claude Opus 4.8 | Claude Opus 4.8 유지 |

## 6. 최종 판단

2회차에서는 Gemini 3.1 Pro 재테스트를 통해 모델 비교의 공정성을 보완했다.

Gemini 3.1 Pro는 Gemini 3.5 Flash보다 개선된 결과를 보였으나, 이번 과업에서는 Claude Opus 4.8이 결정사항, 검토사항, 확인 필요 사항을 가장 안정적으로 분리했다.

따라서 최종 적용 모델은 Claude Opus 4.8로 유지한다.
