# 회의록 — B2B 제조기업 홈페이지 리뉴얼 프로젝트 킥오프

**회의 주제:** B2B 제조기업 홈페이지 리뉴얼 프로젝트 킥오프 회의
**참석 역할:** 클라이언트 담당자 / 프로젝트 매니저 / 기획 담당 / 디자인 담당 / 퍼블리싱 담당 / 개발 담당

---

## 1. 회의 목적

기존 홈페이지의 노후화로 인한 문제(모바일 화면 가독성 저하, 제품 문의 전환율 저조)를 개선하기 위한 홈페이지 리뉴얼 프로젝트의 착수 및 1차 범위 합의.

리뉴얼 1차 목표는 다음 세 가지로 정리됨:

* 회사 신뢰도 개선
* 제품 정보 탐색 편의성 향상
* 문의 접수 증가

---

## 2. 결정사항

* 홈페이지 주요 메뉴를 **회사소개 / 제품소개 / 기술자료 / 납품사례 / 고객문의**로 구성하기로 함.
* **제품소개 페이지**는 제품군별 목록, 상세페이지, 인증서 다운로드 영역을 포함하기로 함.
* **고객문의 페이지** 폼 입력 항목을 이름, 연락처, 이메일, 문의 제품군, 문의 내용으로 구성하기로 함.
* 관리자 페이지의 **1차 개발 범위는 제품 정보 수정 기능만 포함**하고, **인증서 파일 관리 기능은 추가 견적 대상으로 분리**하기로 함(프로젝트 매니저 제안). ※ 인증서 관리 기능의 최종 포함 여부는 클라이언트 회신 대기 중.
* 다음 회의는 사이트맵 초안 도출 이후 진행하기로 함.

> 참고: 퍼블리싱 담당이 제시한 "반응형 기준 + 모바일 우선 작업" 방향은 회의 중 제시된 작업 의견으로, 확정된 결정사항으로 명시되지는 않음(아래 '확인 필요 사항' 참고).

## 4. 확인된 사실

* 클라이언트가 기존 홈페이지의 모바일 가독성 저하 및 제품 문의 전환율 저조를 문제로 제기함.
* 개발 담당: 제품 정보 관리 기능 구현은 가능하나, 인증서 파일 업로드 관리까지 포함할 경우 개발 범위와 일정이 늘어날 수 있음.
* 개발 담당: 문의 폼 접수 내역을 이메일로 발송하는 기능은 기본 범위에 포함 가능함.
* 게시판 데이터 이전 여부는 현재 데이터 양과 구조가 확인되지 않아 회의 중 즉시 확정하지 못함.
* 전체 오픈 목표일은 다음 달 말로 논의되었으나, 최종 일정은 자료 전달일과 추가 기능 범위 확정 후 재산정하기로 함.
* 다국어 페이지 필요성에 대한 의견이 제시되었으나, 언어 범위와 번역 방식은 이번 회의에서 정하지 않음.

---

## 5. 가능한 리스크

> 아래 항목은 회의 메모에 명시된 사실에서 파생되는 추론으로, 확정 사항이 아님.

* **인증서 파일 관리 기능 포함 시** 개발 범위·일정 증가로 전체 오픈 일정에 영향이 발생할 수 있음.
* **게시판 데이터 이전 범위·구조 미확정**으로 인해 작업량 및 일정이 변동될 수 있음.
* **다국어 페이지 도입 시** 추가 기획·디자인·번역 작업으로 범위, 일정, 비용 변동이 발생할 수 있음.
* **클라이언트 자료 전달(다음 주 월요일) 및 기능 범위 회신(금요일)이 지연될 경우**, 사이트맵 → 디자인 → 퍼블리싱 → 개발로 이어지는 후속 단계 전체가 연쇄적으로 지연될 수 있음.

---

## 6. 확인 필요 사항

* 인증서 관리 기능 1차 범위 포함 여부 (클라이언트 회신 대기)
* 인증서 관리 기능 추가 견적(비용·일정) 산정 필요 여부 및 내용
* 기존 게시판 데이터의 양·구조 및 이전 여부
* 다국어 페이지의 대상 언어 범위 및 번역 방식
* 전체 오픈 최종 일정
* 기존 홈페이지 관리자 접속 정보 및 호스팅 정보
* 디자인 시안 / 퍼블리싱 작업의 구체적 마감 기한
* 개발 작업(제품 정보 수정·문의 폼 이메일 발송)의 일정
* 퍼블리싱 "반응형 + 모바일 우선" 방향의 공식 확정 여부

---

## 7. 후속 일정

* **이번 주 금요일** — 클라이언트, 인증서 관리 기능 포함 여부 회신
* **다음 주 월요일** — 클라이언트, 로고 원본·제품 카탈로그·인증서 PDF·납품사례 자료 전달
* **다음 주 화요일** — 기획, 사이트맵 및 주요 화면 구성안 작성 완료
* **사이트맵 확정 후** — 디자인, 메인 시안 1안 제작
* **디자인 확정 후** — 퍼블리싱, 반응형(모바일 우선) 작업
* **사이트맵 초안 도출 후** — 차기 회의 진행
* **전체 오픈 목표(잠정)** — 다음 달 말 (자료 전달일·추가 기능 범위 확정 후 재산정)
