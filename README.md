# B1-1 미션 제출 자료

## 1. 미션 개요

이번 미션에서는 웹에이전시의 B2B 제조기업 홈페이지 리뉴얼 킥오프 회의록 정리 업무를 대상으로, LLM 기반 업무 자동화 프롬프트 패키지를 설계했습니다.

핵심 과업은 회의 메모에서 결정사항, 확인 필요 사항, 후속 실행 항목을 분리하여 실무 문서로 정리하는 것입니다.

|---|---|
| `01_모델비교_선정보고서.pdf` | ChatGPT, Claude, Gemini 모델 비교 및 최종 모델 선정 근거 |
| `02_시스템설계문서.md` | 문제 정의, 입력 템플릿, 페르소나, 시스템 프롬프트, 환각 검증 설계 |

| `logs/` | Claude Opus 4.8 기준 Turn별 실행 로그 원본 |

## 4. 모델 비교 요약

1회차에서는 ChatGPT 5.5, Claude Opus 4.8, Gemini 3.5 Flash를 비교했습니다.

다만 Gemini만 Flash 계열 모델을 사용했다는 점에서 비교 공정성의 한계가 있었습니다.  
이에 2회차에서는 Gemini 3.1 Pro를 추가 테스트하여 상위 모델 간 비교가 가능하도록 보완할 예정입니다.

현재 기준으로는 Claude Opus 4.8이 이번 과업의 핵심인 결정사항, 검토사항, 확인 필요 사항의 분리에서 가장 안정적으로 작동했다고 판단했습니다.

따라서 최종 적용 모델은 Claude Opus 4.8로 유지하되, 2회차에서는 Gemini 3.1 Pro 테스트 결과를 추가 반영하여 모델 선정 근거를 보완합니다.
